Full Stack Deployment course project by Sajjad 

i learn how to deploy a full stack application with this course. 
Course chaicode: https://www.youtube.com/watch?v=vzVbqXVID-Y

## Project Structure

- `Client`: React application with Vite
- `Server`: Node.js server with Express
- `Dockerfile`: Docker configuration for the server
- `docker-compose.yml`: Docker Compose configuration to run the server and client
- `README.md`: Project documentation
